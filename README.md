Multi-Agent Stock Analysis Tool

This project uses a multi-agent AI system powered by Groq (LLaMA-4) and yFinance to provide comprehensive stock market predictions.

🧠 The Agent System

Agent 1 (News Sentiment): Scrapes Yahoo Finance and Finviz to analyze market sentiment.

Agent 2 (Technical Predictor): Evaluates RSI, MACD, SMAs, and Bollinger Bands.

Agent 3 (Master Synthesizer): Fuses all insights into a final "Strong Buy" to "Strong Sell" verdict.

Agent 4 (Insider/Institutional): Analyzes SEC Form 4 filings and smart money flow.

🚀 Getting Started

Clone the repo: git clone https://github.com/your-username/Stock-Analyzer-Project.git

Install dependencies: pip install -r requirements.txt

Configure API Key: Paste your Groq API key into the GROQ_KEY variable within the notebook.

Run the analysis: Call analyze_stock('AAPL') to generate a report.
